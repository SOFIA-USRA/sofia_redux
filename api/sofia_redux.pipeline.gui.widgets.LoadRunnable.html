LoadRunnable¶

class sofia_redux.pipeline.gui.widgets.LoadRunnable(load_function, data, param, dirname)[source]¶

Bases: QRunnable

Worker class to load reduction data.

This class is intended to be the worker for loading reduction data in a thread separate from the main GUI thread. It uses CustomSignals to emit a ‘finished’ signal when the data loading is complete.

Attributes:
loadfunction

Reduction load function to run (redux.Interface.start).

datalist of str

Input data file names.

paramredux.Parameters

Parameters to save and pass to the new reduction object.

dirnamestr

Output directory to save and pass to the new reduction object.
