SkyGradient¶

class sofia_redux.scan.channels.gain_provider.sky_gradient.SkyGradient(horizontal=True)[source]¶

Bases: ZeroMeanGains

Creates a SkyGradient gain provider.

The sky gain provider returns the horizontal or vertical position data with a mean value removed. The mean value is determined from a given mode during validate.

Parameters:
horizontalbool, optional

If True (default), applied to the horizontal position. Otherwise, operates on the vertical position data.
