FORCASTSlitcorrParameters¶

class sofia_redux.pipeline.sofia.parameters.forcast_slitcorr_parameters.FORCASTSlitcorrParameters(default=None, drip_cal_config=None, drip_config=None)[source]¶

Bases: FORCASTSpectroscopyParameters

Reduction parameters for the FORCAST grism slitcorr pipeline.

Initialize parameters with default values.

The various config files are used to override certain parameter defaults for particular observation modes, or dates, etc.

Parameters:
drip_cal_configdict-like, optional

Reduction mode and auxiliary file configuration mapping, as returned from the sofia_redux.instruments.forcast getcalpath function.

drip_configdict-like, optional

DRIP configuration, as loaded by the sofia_redux.instruments.forcast configuration function.

to_config()[source]¶

Read parameter values into a configuration object.

Section names in the output object are written as stepindex: stepname in order to record the order of reduction steps, and to keep any repeated step names uniquely identified. Only the current parameter values are recorded. Other information, such as data or widget type or default values, is lost.

Overrides parent function in order to add a slitcorr = True flag to the top-level configuration.

Returns:
ConfigObj

The parameter values in a configobj.ConfigObj object.
