calchilo¶

Parameters:
choparray-like of float

Chop or nod offset values.

nsampint

Number of samples in the chop array.

choptolfloat

Tolerance for deviation from high or low value.

goodarray-like of int

Flag value for each sample. Must match the dimension of chop. Updated in place.

The high chop value.

choplofloat

chopefffloat

The effective chop value.
