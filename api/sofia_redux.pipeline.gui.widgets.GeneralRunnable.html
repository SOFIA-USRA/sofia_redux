GeneralRunnable¶

class sofia_redux.pipeline.gui.widgets.GeneralRunnable(run_function, *args, **kwargs)[source]¶

Bases: QRunnable

Worker class to run a general function.

This class is intended to be the worker for running any function that does not require specific input or output in a separate thread. It uses CustomSignals to emit a ‘finished’ signal when the function is complete.
